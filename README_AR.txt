
دليل سريع لتشغيل مشروع Ninja Math Runner
=======================================

1) افتح المجلد ninja-math-runner-project.

2) إذا أردت الحفظ الأونلاين:
   افتح الملف firebase-config.js
   وضع بيانات مشروع Firebase مكان YOUR_...

3) إذا لم تضف Firebase الآن:
   اللعبة ستعمل عادي بالتخزين المحلي داخل المتصفح.

4) لتشغيل اللعبة محليًا:
   افتح التيرمنال داخل المجلد ثم نفذ:

   python -m http.server 5500

5) ثم افتح المتصفح على:
   http://localhost:5500

ملاحظات مهمة:
- تم ربط الصور الحقيقية من ملف assets الذي أرسلته.
- اللاعب والأعداء والعملات تعمل بصور متحركة.
- يوجد Level 1 و Level 2.
- الخطر والكنز ونهاية المستوى مرتبطة بأسئلة رياضيات.
- لوحة الصدارة تعرض أفضل 3 طلاب والنتائج الكاملة.

الملفات الأساسية:
- index.html        : الهيكل العام للصفحة
- style.css         : التصميم
- main.js           : منطق اللعبة وPhaser وFirebase
- firebase-config.js: إعدادات Firebase

- تم توحيد التخزين والقراءة من Collection واحدة اسمها players داخل Firestore.


تم الآن إدخال إعدادات Firebase الحقيقية في الملف firebase-config.js.
عند تشغيل المشروع عبر سيرفر محلي أو بعد رفعه على الاستضافة، ستُحفظ النتائج داخل Cloud Firestore في collection اسمها players.
